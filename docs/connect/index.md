# Connecting to CRCD

CRCD provides multiple ways to access computing resources. Choose the method that best fits your workflow.

## Connection Methods

| Method | Best For | Access URL |
|--------|----------|------------|
| [SSH Terminal](../web-portals/terminal.md) | Command-line workflows, scripting | `ssh username@htc.crc.pitt.edu` |
| [Open OnDemand](../web-portals/open-ondemand.md) | Web-based job management, file browsing | [ondemand.htc.crc.pitt.edu](https://ondemand.htc.crc.pitt.edu) |
| [JupyterHub](../web-portals/jupyter-hub.md) | Interactive notebooks, data analysis | [hub.crc.pitt.edu](https://hub.crc.pitt.edu) |
| [VIZ Desktop](../web-portals/viz.md) | GUI applications, visualization | [viz.crc.pitt.edu](https://viz.crc.pitt.edu) |
| [VS Code](../slurm/vscode.md) | Remote development, debugging | Via SSH extension |

## Quick Start

1. Ensure you are connected to the [Pitt VPN](https://services.pitt.edu/TDClient/33/Portal/KB/ArticleDet?ID=293)
2. Choose an access method from the table above
3. Log in with your University of Pittsburgh credentials

## Specialized Portals

These portals are available for specific research groups:

- [Psych Portal](../web-portals/psych.md) - Psychology department resources
- [HUGEN Portal](../web-portals/hugen.md) - Human Genetics department resources

## Additional Resources

- [JupyterHub on Teach Cluster](../web-portals/jupyter-teach.md) - For classroom instruction
- [Jupyter on OnDemand](../web-portals/jupyter-ondemand.md) - Jupyter via the OnDemand portal
- [JupyterHub Virtual Environments](../web-portals/jupyterhub-venv.md) - Custom Python environments in JupyterHub

!!! tip
    For most users, **Open OnDemand** provides the easiest starting experience with a web-based interface for file management, job submission, and interactive applications.
