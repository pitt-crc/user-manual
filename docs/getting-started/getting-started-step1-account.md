# Step 1: Getting an Account

Access to the CRCD ecosystem requires a CRCD account and a resource allocation.

## Who Can Request an Allocation?

All active Pitt faculty, instructors, or center directors can request a resource allocation using the webform from the [service catalog](https://crc.pitt.edu/service-request-forms).

## Network Access

The CRCD ecosystem is firewalled within PittNet. You must establish a [VPN connection](https://services.pitt.edu/TDClient/33/Portal/KB/ArticleDet?ID=3426) before accessing any CRCD resources.

!!! tip "Alternative: Azure Virtual Desktop"
    If the VPN is non-functional after exhausting all troubleshooting, you can use an [Azure Virtual Desktop](access_avd.md) as a jump box to CRCD.

    **CRCD users on UPMC devices** should follow these [instructions](upmc_avd.md) for connecting via the HSIT-AVD instance.

!!! tip "Accessing the CRCD Secure Research Environment (SRE)"
    The CRCD SRE can be accessed using a secured [Azure Virtual Desktop](access_sre.md). Only projects with an IRB approval or a data user agreement (DUA) requiring NIST SP 800-171 compliance can reside within this restricted environment.

!!! warning "HIPAA Environment Deprecation"
    The HIPAA environment has been upgraded to the NIST SP 800-171 standard (self-attested). Use the SRE instructions above instead.

## Definitions

| Term | Description |
|------|-------------|
| Resource allocation | An allotment of computing time and/or data storage quota |
| Client | Your computer or internet-connected device |
| PittNet | The internal University network |
| Firewall | A network security device that filters traffic based on security policies |
