# Big Picture Overview

The University of Pittsburgh provides its research community access to high-performance computing and data storage resources, maintained and supported through the Center for Research Computing and Data (CRCD) and Pitt IT.

To get started, you need a CRCD account, which you use to log in to Access Portals and interact with the CRCD Ecosystem.

![GETTING-STARTED-MAP](../_assets/img/getting-started/getting-started-map.png)

## Who Has Access?

The [CRCD-P3: Open Science Research Computing and Data Environment Usage Policy](https://crc.pitt.edu/sites/default/files/assets/CRC%20P3%20Open%20Science%20Research%20Computing%20and%20Data%20Environment%20Usage%20Policy.pdf) defines the scope of access:

- All Pitt faculty, instructors, Emeritus faculty, and center directors
- Students, postdoctoral fellows, and staff sponsored by eligible PIs
- Alumni and external collaborators via the [Sponsored Account](sponsored_account.md) mechanism

## Available Clusters

| Cluster | Full Name | Use Cases |
|---------|-----------|-----------|
| [MPI](../hardware_profiles/mpi.md) | Message Passing Interface | Tightly coupled parallel codes distributed across multiple nodes |
| [HTC](../hardware_profiles/htc.md) | High Throughput Computing | Genomics and health sciences workflows on single nodes |
| [SMP](../hardware_profiles/smp.md) | Shared Memory Processing | Single-node jobs with shared memory parallelism |
| [GPU](../hardware_profiles/gpu.md) | Graphics Processing Unit | AI/ML and GPU-accelerated simulations |

## Definitions

| Term | Description |
|------|-------------|
| Client | Your computer or internet-connected device |
| Access Portal | A remote server used to submit jobs or manage data |
| CRCD Ecosystem | The total infrastructure: clusters, storage, networking, and software |
