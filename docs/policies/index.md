This section documents the policies governing the use of CRCD computing resources, including job scheduling, resource allocation, and hardware investment opportunities.

## Key Points

!!! important "Resource Limits"
    Each user and group is subject to resource limits that ensure fair access across the research community. See the [Job Scheduling Policy](job-scheduling-policy.md) for current limits on each cluster.

!!! note "Proposal Support"
    If you are writing a grant proposal that includes CRCD resources, see our [resource descriptions page](resource-descriptions-for-writing-proposals.md) for recommended language and current system specifications.
