# Help and Support

CRCD provides multiple channels for getting help with your research computing needs.

## Getting Help

- **Help Portal**: Submit a ticket at [services.pitt.edu](https://services.pitt.edu/TDClient/33/Portal/Requests/ServiceDet?ID=1511)
- **Office Hours**: Check [crc.pitt.edu](https://crc.pitt.edu) for current office hours schedule
- **Email**: Contact the CRCD support team for general inquiries

## Common Resources

- [Frequently Asked Questions](../faq.md) - Solutions to common problems
- [Code Optimization Support](../code-optimization.md) - Get help optimizing your research code
- [Acknowledging CRCD](../acknowledge-crc.md) - How to credit CRCD in publications

## Tips and Tricks

- [Persisting Sessions](../tips/persist-sessions.md) - Keep your sessions alive with tmux and nohup

## Contributing to This Documentation

This documentation is maintained on [GitHub](https://github.com/pitt-crc/user-manual). If you find errors or want to suggest improvements, you can:

1. Click the edit icon (pencil) on any page to suggest changes
2. [Open an issue](https://github.com/pitt-crc/user-manual/issues/new) to report problems
3. Submit a pull request with your proposed changes

!!! info "CRCD welcomes your contributions"
    This documentation is open source and maintained collaboratively. Your feedback helps improve the experience for all users.
