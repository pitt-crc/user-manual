# CRCD Technical Documentation

The [University of Pittsburgh Center for Research Computing and Data (CRCD)](https://crc.pitt.edu/) provides high-performance computing, data storage, and research support services to the University of Pittsburgh research community.

## Top Documentation Pages

* [Getting Started at CRCD](getting-started.md) - Information for new and existing CRCD users
* [Getting Help](help/index.md) - How to get support
* [Job Scheduling Policy](policies/job-scheduling-policy.md) - Compute job charge factors, run limits, submit limits
* [Running Jobs](jobs/index.md) - Slurm commands, job script basics, submitting jobs
* [Connecting to CRCD](connect/index.md) - SSH, Open OnDemand, JupyterHub, and more
* [Managing Data](data-management/index.md) - File systems, transfers, and data management
* [Working with Software](applications/index.md) - Modules, Python, R, containers, and licensed software
* [Globus](data-management/file-transfer-methods/globus.md) - High-performance data transfers
