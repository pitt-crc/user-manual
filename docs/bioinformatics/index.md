# Bioinformatics and Domain Support

CRCD provides specialized support for bioinformatics, genomics, imaging, and other domain-specific research workflows. Our team offers consultation, training, and pre-configured pipelines to accelerate your research.

## Support Areas

### Genomics and NGS Analysis

Pre-configured workflows and guidance for next-generation sequencing data analysis:

- [RNA-Seq](../advanced-genomics-support/RNASeq-data-analysis.md) - Differential gene expression analysis
- [Variant Calling](../advanced-genomics-support/variant-calling.md) - SNP and indel detection
- [Single-Cell RNA-Seq](../advanced-genomics-support/single-cell-rnaseq.md) - Single-cell transcriptomics
- [Spatial Transcriptomics](spatialtranscriptomics.md) - Spatially-resolved gene expression
- [ChIP-Seq](../advanced-genomics-support/ChipSeq-data-analysis.md) - Chromatin immunoprecipitation
- [ATAC-Seq](../advanced-genomics-support/ATACSeq-data-analysis.md) - Chromatin accessibility
- [Methylation](Methylation.md) - DNA methylation analysis
- [nf-core Pipelines](nf-core-genomics-pipelines.md) - Community-curated Nextflow pipelines

### AI and Machine Learning

- [AlphaFold3](alphafold3.md) - Protein structure prediction
- [Ollama](ollama.md) - Local LLM inference on GPU nodes
- [GitHub Copilot](copilot.md) - AI coding assistant in RStudio

### Imaging and Visualization

- [Bioimage Analysis](bioimage.md) - QuPath, Napari, and Fiji workflows
- [VirtualGL](virtualgl.md) - Hardware-accelerated remote visualization
- [fMRI Analysis](fMRI.md) - Functional MRI data processing

### Health Data

- [Electronic Health Records](EHR.md) - Working with EHR data on the cluster
